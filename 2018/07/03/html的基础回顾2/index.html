带结构的表格

若表格较为复杂的情况下，为了提高效率，让表格一边加载一边显示。且以下三个标签通常都是同时存在同时使用的。
不影响布局，但是划分了表格的结构。

  1. thead 表格的头

  2. tbody 表格的主体

  3. tfoot 表格的注脚

表格中的常用属性

table标签

  1. width 表格的宽度。

  2. align 表格相对周围元素的对齐方式。

  3. border 设置表格边框的宽度。

  4. bgcolor 表格的背景颜色。

  5. cellpadding 单元边缘与其内容之间的空白。

  6. cellspacing 单元格与单元格之间的空白。

  7. frame 规定外侧边框的哪个部分是可见。

  8. rules 规定内侧边框的哪个部分是可见。

tr标签

  1. align 行内容的水平对齐。

  2. valign 行内容的垂直对齐。

  3. bgcolor 行的背景色。

td标签

  1. 跨列属性colspan

  2. 跨行属性rowspan
