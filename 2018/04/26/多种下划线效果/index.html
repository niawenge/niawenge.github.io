多种下划线效果

发表于 2018-04-26   |   更新于 2018-07-04 | 分类于 CSS

纯CSS实现的一些效果记录。

效果1

效果：鼠标悬浮在内容上，下划线从左到右“展开”出现。鼠标挪开，下划线“缩回”左边。

原理：用before选择器“做出”下划线，定位在内容下方，且初始宽度设置为0。通过transition来改变宽度为100%从而实现动画效果。

效果2

效果：鼠标悬浮在内容上，下划线从中心位置同时向左向右“展开”，鼠标离开则下划线两头“缩回”中心点。

原理：设置宽度100%,然后利用transform: scale() 分别设置参数0和1实现线条运动。

效果3

效果：鼠标悬浮在内容上的时候，下划线从左端出现，鼠标离开时，下划线从右边离开。即“左进右出”的假象。

原理：其实同效果2只是增加了transform-origin，来更改一个元素变形（transform）的原点。即更改scaleX缩放的点。
