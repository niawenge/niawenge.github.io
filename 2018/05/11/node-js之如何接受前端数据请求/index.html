node.js之如何接受前端数据请求

发表于 2018-05-11   |   更新于 2018-05-12 | 分类于 node.js

前端向后端发送请求的通常有这几种方式：form,ajax,jsonp等等
但是对于后端收到的请求性质都是一样的，都是通过http得到的请求。

请求方式：

  1. GET: 数据在URL中,用url模块解析。
  2. POST: 数据不在URL中,用querystring模块解析。相对于“头”，“身体”更大，需要分块解析。
